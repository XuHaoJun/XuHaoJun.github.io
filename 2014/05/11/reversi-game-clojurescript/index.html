Clojurescript 使用心得

一堆括號阿！！每次都要讓函數回傳有意義的值，Debug 起來也很容易，一個函數影響的範圍就只有他的參數而已，沒有隱示參數(全域變數、成員變數)的話就很好測試，每個函數就盡量小小的做一件事，不過到使用 phaser (javascript html5 game framework)那層就一堆副作用囉!(set! xxx yyy)。
